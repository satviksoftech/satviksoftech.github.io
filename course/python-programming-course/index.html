Python For Machine Learning

  • DURATION

    01 Month

  • WEEKLY

    12 hours

  • FEE

    ₹5000

This is the python course which will help you to kick start your career as a software developer. In this course we teach the fundamentals of computer programming through python programming language. You will learn to think logically and improve your problem solving skills, thanks to our revolutionary LEARN WITH LIVE CODING approach. We begin with representing and storing data using python data types and variables and gradually learn how to use python's build in data structures like lists, tuples, dictionaryies etc. Along the way we will solve many puzzles and also understand object oriented programming concepts. We also cover data analysis with numpy, matplotlib, seaborne, pandas. To summarize, this python coding class will take you from an absolute noob to a good developer within a month.

Week 1: Getting our feet wet
  • Undestanding Interpreted languages, Compiled languages
  • Using the Python REPL
  • Decision making statements and problem solving- if-else
  • Loops- while and for with problem solving
  • Strings - Slicing, iteration, Basic string problems using loops
Week 2: Becoming a good coder
  • Understanding Functions
  • Solving problems using functional programming
  • Using functions as first class objects
  • Some built in functions
  • Lists,dictionaries, sets and tuples- Basic usage, using functions
Week 3: Becoming a professional coder
  • Solving problems using Object Oriented Programming
  • Multilevel and multiple inheritance
  • Method overriding and overloading
  • Comprehensions, generators, iterators, file handling, multi threading
  • Building games and apps with tkinter, pygame etc
Week 4: Becoming a Data Scientist
  • Numerical processing in Python
  • Numpy, matplotlib, pandas, seaborn
  • Project-Data analysis with pandas
  • Basic of Machine learning - sklearn library
  • Project - Building a Machine Learning project
Week 5: Becoming a Software Developer
  • Version control with Git
  • Building web applications
  • Flask web app framework basics
  • SQL - Database design and queries
  • Project - Building a web application with database support using Flask
