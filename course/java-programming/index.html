Java Application Programming

  • DURATION

    01 Month

  • WEEKLY

    12 hours

  • FEE

This is a unique Java course which takes student from novice to professional android developer all within a single month. Our unique hands-on approach coupled with focus on OOP strategies to solve problems lays down strong foundation for advanced Java concepts. This is must have course for any student wishing to truly understand OOP concepts. Java is a very popular language in which Object Oriented Programming is heavily relied upon. We focus on unique aspects of Java right from first class so that students are able to think the Java way right from the beginning. The course also focuses on reading official documentation so that students also learn how to learn on their own. The course ends with Android App Development using Java

Week 1: Introduction, data types, loops
  • History of java, jdk, jvm, current versions
  • Data types
  • Decision making statements and problem solving- if-else, switch
  • Loops- while and for with problem solving
  • Arrays- problem solving with arrays
Week 2: Becoming a good coder
  • Understanding Functions and scope
  • Solving problems using functional programming
  • Static keyword and its usage
  • Polymorphism
Week 3: Becoming a professional coder
  • Multilevel and multiple inheritance
  • Method overriding and overloading
  • Abstract classes, interfaces
  • Solving problems using Object Oriented Programming
  • Strings, wrapper classes, Generics
Week 4: Path to Java expertise
  • Generic interfaces - Comparator and Comparable
  • Collections in Java
  • Solving problems using collections
  • Building desktop apps with Swing
  • Java 8 streams and lambda functions
  • File Handling, Exception handling, multi threading
Week 5: Becoming a App Developer
  • JDBC and Sql
  • Building web applications using JSP
  • Android SDK, Activity, Activity life cycle
  • Gradle, Implementing android apps
  • Project - Building a android app
